## Customizing Shortcuts

### How to Customize

```
1. Open Settings (Ctrl+,)
2. Go to: Advanced → Keyboard Shortcuts
3. Find action to customize
4. Click on current shortcut
5. Press desired key combination
6. Confirm with Enter
7. Restart application (if needed)
```

### Reset Shortcuts

```
Method 1: Full Reset
1. Settings → Advanced → Keyboard
2. Click "Reset All to Defaults"
3. Restart application

Method 2: Individual Reset
1. Settings → Advanced → Keyboard
2. Find shortcut to reset
3. Click "Reset to Default"
```

### Unsupported Customizations

```
Cannot customize:
- System shortcuts (Alt+F4, Win+X, etc.)
- Accessibility shortcuts
- BIOS/firmware keys
- Reserved OS keys

Can customize:
- Nearly all application shortcuts
- View shortcuts
- Tool shortcuts
- Simulation shortcuts
```

## Troubleshooting Shortcuts

### Shortcuts Not Working

```
Solution 1: Check Modifiers
- Correct key order matters
- Ctrl+S not Ctrl Pressed then S
- Some keys may be OS-reserved

Solution 2: Check Caps Lock
- Shift modifiers interact with Caps Lock
- Sometimes causes unexpected behavior

Solution 3: Keyboard Layout
- Non-US layouts may affect shortcuts
- Remap if needed in Settings

Solution 4: Restart Application
- Shortcut changes require restart
- Close all windows and reopen

Solution 5: Reset Shortcuts
- If customized incorrectly
- Reset to defaults in Settings
```

### Conflicting Shortcuts

```
Problem: Shortcut does nothing
Cause: Conflicting with system shortcut

Solution:
1. Check if OS uses shortcut
2. Change in: Settings → Keyboard
3. Choose different key combo
4. Restart app
```

### Key Not Responding

```
Check:
1. Is Key Locked (Caps, Num Lock)?
2. Is Modifier held correctly?
3. Are other keys interfering?
4. Try different keyboard

If Still Issues:
- Report bug with: Ctrl+Alt+R (Report)
- Include keyboard layout info
- Include OS version
```

## Accessibility Shortcuts

### For Users with Motor Impairments

```
Sticky Keys:
- Enables one-handed operation
- Shortcut: Press Shift 5 times
- Windows Setting: Ease of Access

Toggle Keys:
- Audio feedback for locks
- Helps verify key press
- Windows Setting: Ease of Access

Mouse Keys:
- Control with numeric keypad
- Allows keyboard-only operation
- Windows Setting: Ease of Access
```

### For Users with Visual Impairments

```
High Contrast:
- Ctrl+Alt+C (in app)
- Windows Setting: Display → High Contrast

Screen Reader:
- Ctrl+Alt+R (toggle support)
- Use: NVDA (Windows), VoiceOver (Mac)
- All shortcuts announced

Tab Navigation:
- Tab: Next element
- Shift+Tab: Previous
- Follow logical order
- Works throughout app
```

### For Users with Cognitive/Learning Differences

```
Simpler Shortcuts:
- Fewer modifiers if possible
- Consistent patterns
- Documented with visuals

Feature:
- Settings → Accessibility → Simplified Mode
- Reduces shortcut complexity
- Hides advanced features
```
