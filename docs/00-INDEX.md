**Version:** 0.4.0
**Framework:** .NET 8, WinUI 3, Windows 10 1809+
**Producer:** Hamed Rezaei

### Key Features

- 2D weave design with infinite canvas
- 2D simulation with continuous yarn ovals
- 3D simulation with multiple yarn path models
- 27 built-in weave patterns
- Model builder with AI assistance
- Advanced fabric physics simulation
